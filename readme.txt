Gallereo plugin updates standard gallery with fancy Google+ like layout and nice preview.

== Description ==

Gallereo plugin updates standard gallery with fancy Google+ like layout and nice preview. The plugin is very easy in use, doesn't require any configuration, works from the box.

== Installation ==

1. Go to **Plugins** Â» **Add New** page;
1. Click on **Upload** tab;
1. Select the plugin archive in the file input field and click **Install Now** button;
1. After the plugin will be installed, activate it;
1. Once activated, installation is finished.

After you've installed the plugin, create or edit a post and add a gallery there. The plugin automatically will update your gallery with a nicer view.
